# 파이썬 기초 샘플 코드 (Python Basics)

이 폴더에는 파이썬 학습을 위한 5개의 기초 파일이 포함되어 있습니다.

## 파일 목록

1.  **[01_basics.py](01_basics.py)**: 변수, 데이터 타입, 출력 및 기본 연산
2.  **[02_conditionals.py](02_conditionals.py)**: if/elif/else 조건문 사용법
3.  **[03_loops.py](03_loops.py)**: for 반복문과 while 반복문
4.  **[04_functions.py](04_functions.py)**: 함수 정의, 매개변수, 반환값(return)
5.  **[05_data_structures.py](05_data_structures.py)**: 리스트(List)와 딕셔너리(Dictionary) 기초

## 실행 방법

각 파일은 파이썬이 설치된 환경에서 터미널(명령 프롬프트)을 통해 실행할 수 있습니다.

```bash
python 01_basics.py
python 02_conditionals.py
python 03_loops.py
python 04_functions.py
python 05_data_structures.py
```

`02_conditionals.py`는 실행 시 점수를 입력해야 결과를 볼 수 있습니다.
